# Simple Python Project

这是一个简单的 Python 项目，目的是展示如何创建一个开源项目并使用开源许可证。

## 功能
- 打印"Hello, World!"。

## 安装
1. 克隆项目到本地：
    ```bash
    git clone https://github.com/yourusername/simple-python-project.git
    ```
2. 进入项目目录：
    ```bash
    cd simple-python-project
    ```
3. 运行 `main.py`：
    ```bash
    python main.py
    ```
